# iOSTemplates

## Usage

iOSTemplates is an iOS app, with 3 template modules:
- [MVC](#mvc-template)
- [MVVM](#mvvm-template)
- [VIPER](#viper-template)

### MVC Template
- Model
    - Data modeling
- ViewController
    - Lifecycle
    - Rendering
    - API Requests (mocked)
    - Data processing
    - Navigation

### MVVM Template
- Model
    - Data modeling
- ViewController
    - Lifecycle
    - Rendering
- ViewModel
    - API Requests (mocked)
    - Data processing
    - Navigation


### VIPER Template
- Entity
    - Data modeling
- ViewController
    - Lifecycle
    - Rendering
- Interactor
    - API Requests (mocked)
- Presenter
    - Data processing
- Router
    - Navigation

## Instalation

Download the project, and:

```bash
$ pod install
```

Then run the app in Xcode.
